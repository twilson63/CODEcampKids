Inline vs. Block Elements

Match the Arrangement

Take a look at the following image. You can see colored bars arranged in a specific fashion. The goal of this exercise is to generate this layout by organizing the various inline and block level elements commented out in the markup.

You have two types of block-level elements that you can use: p and div It does not matter which you choose to use.

You have one type of inline element: span

Finally, you can use a <br /> tag to skip to a new line (it stands for break).

Example #1

Look in the markup at the top of this document and copy/paste the structural elements into the div below. Save the file and refresh your browser to see the results. Keep experimenting with using span, p, and div elements until you achieve the proper layout.

Remember, block level elements take up as much space as the container element they are in. Inline elements take up only the space of their content.

Note: do not worry about small differences in the white space surrounding each color bar. You will learn how to adjust these things using CSS in the next class.
