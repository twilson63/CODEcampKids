CODEcamp Kids Cyber Academy Training

Class 1

Class Overview
Focus: General HTML
Key Topics
  • What is HTML?
  • How is it read by the browser?
  • The DOM
  • Separation of Concerns
    • Content = HTML
    • Presentation = CSS
    • Functionality = Javascript
  • Key Components of an HTML Document
    • Doctype Declaration
    • html Encapsulation
    • Head Section
    • Body Section
  • General HTML Syntax
    • Opening/Closing Tags
    • Self-Closing Tags (e.g., img, input, link)
  • Inline versus Block Elements
  • Intro to SublimeText and Chrome Web Tools
  • Importance of Indentation and Formatting
  • ID's and Classes: Importance of Semantics
  • Discussion of Elements versus Tags versus Attributes
Objectives
Students should...
  • Have a clear understanding of what HTML is and how it is interpreted by the browser.
  • Understand key components of the DOM.
  • Inspect the DOM hierarchy and view code source on any web page.
  • Understand the key differences between HTML, CSS, and JavaScript.

Class 2

Class Overview
Focus: Basics of CSS, Introduction to the Box Model
Key Topics
  • Review of Last Class
  • What is CSS?
    • Inline Styles versus Inline Stylesheets versus External Stylesheets
  • How to Create and Link to External Stylesheets
  • Basic CSS Selectors
    • How to Select Entire Elements (e.g., div, p, body, etc.)
    • Basic Syntax
    • How to Target Specific Elements
  • The Box Model
  • Basic CSS Properties (color,background-color, font-style, etc.)
  • CSS Resets
Objectives
Students should...
  • Understand the basics of CSS selectors.
  • Understand the basics of the Box Model and how to calculate dimensions of HTML elements.
  • Understand the importance of CSS resets.
  • Understand differences between Classes and ID's.

Class 3

Class Overview
Focus: More CSS Properties, Beginning of a Basic Layout
Key Topics
  • Review Box Model and Display Property
  • display: inline-block
  • Building a Header
  • Floats
    • Importance of When/Where To Use Floats, and When Not To Use Them
    • Discussion of How Content Flows Around Floated Elements
    • Clearing Floats
  • Live In-Class Development of a Basic Layout
Objectives
Students should...
  • Have a thorough understanding of inline vs block level elements.
  • Understand the basics of floats and the caveats they present.
  • Understand the key concepts behind wrappers/container elements.
  • Understand the importance of formatting code and keeping it semantic for readability.

Class 4

Class Overview
Focus: More CSS and HTML
Key Topics
  • Higher-Level Discussion of "Wrappers" and "Containers"
  • IMG Tag and Linking to Local Images
  • More CSS Properties
    • background-image
    • border-radius
    • margin: auto
    • text-align and text-decoration
  • Psuedo Classes
    • :hover
    • first-child, last-child, and nth-child()
    • direct child selector >
    • sibling selector +
  • CSS Positioning
  • Creating a Basic Layout From Everything We Have Learned
  • REVIEW
Objectives
Students should...
  • Have a base knowledge of many common CSS properties.
  • Understand differences between static, relative, and absolute positioning.
  • Have a basic knowledge of more advanced CSS selectors.
  • Have the ability to build a basic HTML layout and style it with CSS.
